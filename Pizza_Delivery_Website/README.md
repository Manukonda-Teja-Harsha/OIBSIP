# Domino-Style Pizza Delivery

This workspace contains a full-stack pizza delivery app with React/Tailwind frontend and Node.js/Express backend.

## Setup

1. Backend:
   - `cd backend`
   - `npm install`
   - Copy `.env.example` to `.env` and configure values
   - `npm run dev`

2. Frontend:
   - `cd frontend`
   - `npm install`
   - `npm start`

## Notes

- Backend API runs on `http://localhost:5000`
- Frontend runs on `http://localhost:3000`
- The app uses JWT auth, email verification, and a Domino-inspired UI
